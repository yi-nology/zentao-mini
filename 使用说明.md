# 禅道Mini 使用说明

## 1. 应用简介

禅道Mini 是一款基于 Wails 和 Vue 开发的禅道项目管理系统辅助工具，提供了 Bug 查询、需求查询、任务查询和工时统计等核心功能，帮助团队更高效地管理项目。

## 2. 系统初始化

首次使用禅道Mini 时，需要进行系统初始化。

### 2.1 初始化步骤

1. **打开应用**：启动禅道Mini 应用
2. **上传配置文件**：在初始化引导页面，点击 "选择文件" 按钮，上传使用 `generate-encryption.sh` 脚本生成的 `auth-config.json` 文件
3. **开始初始化**：点击 "开始初始化" 按钮
4. **初始化完成**：系统会自动跳转到主页面
5. **测试连接**：可选择点击 "测试禅道连接" 按钮，验证与禅道系统的连接状态

## 3. 主界面布局

禅道Mini 的主界面分为三个主要部分：

1. **左侧导航栏**：包含各个功能模块的入口
2. **顶部栏**：显示当前页面标题和产品/项目选择器
3. **主内容区**：根据选择的功能模块显示相应内容

## 4. 功能模块使用

### 4.1 产品与项目选择

在使用任何功能模块前，需要先选择产品和项目：

1. **选择产品**：在顶部栏的产品选择器中选择一个产品
2. **选择项目**：产品选择后，项目选择器会自动加载该产品下的项目，选择一个项目

![产品项目选择](https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=A%20screenshot%20of%20the%20product%20and%20project%20selector%20in%20Zentao%20mini%20application%2C%20showing%20dropdown%20menus%20for%20selecting%20product%20and%20project%2C%20clean%20modern%20UI%20design&image_size=landscape_4_3)

### 4.2 Bug 查询

Bug 查询模块允许您查看和筛选 Bug 列表。

#### 功能特点：
- **多维度筛选**：支持按指派人、状态、时间范围筛选
- **详细信息**：查看 Bug 的详细信息，包括标题、状态、严重程度、指派人等
- **导出功能**：支持导出选中的 Bug 为 Excel 文件
- **禅道链接**：点击 Bug 标题可直接打开禅道中的对应 Bug 页面

#### 使用步骤：
1. 在左侧导航栏点击 "Bug 查询"
2. 选择产品和项目
3. 设置筛选条件（可选）
4. 点击 "查询" 按钮
5. 查看 Bug 列表
6. 可选择 Bug 后点击 "查看详情" 或 "导出"

### 4.5 工时统计

工时统计模块提供了直观的工时数据统计和分析。

#### 功能特点：
- **快捷时间范围**：支持本周、上周、本月、上月的快捷选择
- **多维度筛选**：支持按执行/迭代、用户、时间范围筛选
- **数据可视化**：提供每日工时、项目分布、任务类型分布的图表
- **工时明细**：显示详细的工时流水记录
- **数据缓存**：支持数据缓存，提高查询速度

#### 使用步骤：
1. 在左侧导航栏点击 "工时统计"
2. 选择产品
3. 选择时间范围（可使用快捷按钮）
4. 设置其他筛选条件（可选）
5. 点击 "查询统计" 按钮
6. 查看统计结果和图表
7. 可在工时流水明细表中搜索和排序

## 5. 常见操作

### 5.1 导出数据

在 Bug 查询模块中，您可以导出选中的 Bug 为 Excel 文件：

1. 在 Bug 列表中选择一个或多个 Bug
2. 点击 "导出" 按钮
3. 系统会生成并下载 Excel 文件

### 5.2 查看禅道原始页面

在 Bug、需求、任务列表中，点击标题链接可以直接打开禅道中的对应页面：

1. 在列表中找到需要查看的项
2. 点击标题链接
3. 系统会在浏览器中打开禅道的对应页面

### 5.3 重置筛选条件

在各个查询模块中，您可以重置筛选条件：

1. 点击 "重置" 按钮
2. 所有筛选条件会被清空

## 6. 系统要求

- **操作系统**：支持 Windows、macOS、Linux
- **禅道版本**：兼容禅道 12.0 及以上版本
- **网络连接**：需要与禅道服务器保持网络连接

## 7. 故障排除

### 7.1 初始化失败

- **原因**：可能是配置文件不正确或网络连接问题
- **解决方法**：检查配置文件是否正确，确保网络连接正常，重新尝试初始化

### 7.2 数据加载失败

- **原因**：可能是网络连接问题或禅道服务器响应缓慢
- **解决方法**：检查网络连接，稍后重试

### 7.3 导出失败

- **原因**：可能是数据量过大或浏览器兼容性问题
- **解决方法**：减少选择的项数量，或尝试使用不同的浏览器
